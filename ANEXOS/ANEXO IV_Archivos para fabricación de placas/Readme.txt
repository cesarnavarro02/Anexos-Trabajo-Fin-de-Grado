En este Anexo se encuentran los archivos del esquemático y de el footprint de las placas junto con la librería con los
componentes que se han diseñado. Para utilizar estos archivos es necesario tener instalado el software Eagle en su 
versión 7.5. Se debe crear un nuevo proyecto donde se importarán tanto el esquemático como el footprint de la placa 
que se desee fabricar. Además se debe importar la librearía que contiene con nombre CESAR.lbr, que contiene los componentes
específicos de las placas.
